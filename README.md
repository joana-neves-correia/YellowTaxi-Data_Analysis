# YellowTaxi - NYC Taxi Data Analysis Project
Data analysis project using NYC Yellow Taxi records to explore trip patterns, fares, time trends, and geospatial insights through cleaning, transformation, and visualization.

## License
This project is for educational purposes. Feel free to use and adapt the dataset for learning and analysis.

## Overview

This project analyzes the NYC Yellow Taxi dataset to understand trip behavior, pricing patterns, temporal trends, and geographical distribution. The work includes data cleaning, feature engineering, exploratory analysis, and visualizations providing insights into taxi operations across New York City.

## Objectives
- Clean and prepare the Yellow Taxi dataset for analysis.
- Explore trip duration, distance, fares, and passenger patterns.
- Identify temporal trends (hourly and dayly).
- Analyze spatial behavior using pickup locations.
- Generate visual insights supporting operational and business decisions.

## Key Features
- Data cleaning (removing invalid records, correcting datatypes).
- Creation of additional metrics such as fare per mile and peak-hour indicators to support analysis.
- Time-series analysis of trips and revenue.
- Geospatial mapping of pickup zones.
- Visual dashboards highlighting trends and KPIs.
